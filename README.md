# Collection Quest V.02

## Prototype Build — Hack & Slash Development (Step 1)

**Collection Quest V.02** is an early-stage prototype project and the first major step in the development of a larger indie hack-and-slash action game.

This repository represents a **foundational systems build**, focused on iterating quickly, validating core gameplay loops, and establishing a scalable architecture for future development.

---

## About This Project

This project is being developed independently by an indie game developer as part of an ongoing effort to build a complete action RPG / hack-and-slash experience from the ground up.

The focus of this stage is not content depth, but **systems clarity, responsiveness, and extensibility**.

---

## Current Development Goals

* Core player movement and responsive controls
* Basic melee combat and interaction loops
* Enemy behavior prototypes and combat testing
* State-driven architecture for gameplay flow
* Modular system design for future scaling (abilities, enemies, upgrades)

---

## Project Status

This is a **prototype / pre-production build**.

Expect:

* Frequent refactors
* Experimental systems
* Placeholder assets and logic
* Rapid iteration of mechanics

Nothing here is final.

---

## Tech Stack

* Unity Engine
* C#
* Unity Input System
* Custom state machine architecture
* Modular gameplay systems (abilities, effects, and interactions)

## Development Philosophy

This project is being built with a systems-first mindset:

* Keep mechanics modular and reusable
* Avoid over-engineering early content
* Prioritize gameplay feel and iteration speed
* Build toward a scalable hack-and-slash framework
